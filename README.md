# Java Calculator
This calculator runs in the console.
```
java MyCalculator.java
```
## Features
### Standard Mode
<ul>
  <li>Addition</li>
		<li>Subtraction</li>
		<li>Multiplication</li>
		<li>Division</li>
</ul>

### Scientific Mode
<ul>
  <li>Addition</li>
		<li>Subtraction</li>
		<li>Multiplication</li>
		<li>Division</li>
		<li>Sine</li>
  <li>Cosine</li>
  <li>Tangent</li>
</ul>
